Experiment 6: DC Motor

Aim: Develop a ESP32 program to control DC motor in clockwise and anticlockwise direction.
void setup() {
// put your setup code here, to run once:

pinMode(33,OUTPUT);
pinMode(35,INPUT);
}
void loop() {
// put your main code here, to run repeatedly:
if(digitalRead(35))
{

}
else
{
